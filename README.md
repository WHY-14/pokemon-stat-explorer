# Pokemon Stat Explorer
My first ever project on GitHub. 
An interactive Pokemon data analysis tool with dashboard built with Dash and Plotly.

## Features
- Scatter plot with visualised Pokemon distribution.
- Filter by type, generation, and BST range.
- Pokemon information card to view by clicking.
- BST Top 10 / Bottom 10 ranking tables.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python explorer.py
```

Then open your browser and go to the link in the terminal.

## Tech Stack
- Python
- Dash / Plotly
- Pandas
- SQLite
- Dash Bootstrap Components
